# realtyprice

분양권 매각판단 대시보드 MVP입니다.

## 목표

사용자가 등록한 대상아파트의 분양권 매각 판단을 위해 비교단지 실거래가, 전세가, 매매호가, 매물소진추정, 동호수 등급 보정을 반영해 예상 매매가, 예상 전세가, 권장 매각호가, 방어가격, 상승가능성 점수를 산출합니다.

## 실행

```bash
npm install
npm run dev
```

## 주요 원칙

- 백엔드 없음
- localStorage 기반 저장
- 지도 API 미사용
- 네이버부동산/호갱노노 외부 링크 제공
- 관심아파트 별도 개념 없음: 사용자가 추가하는 아파트는 모두 대상아파트
